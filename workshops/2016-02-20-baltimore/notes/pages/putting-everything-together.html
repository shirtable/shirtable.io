with open(filename, "r") as f:

    # Just read the first line because we know its the column descriptions.
    cols = f.readline()
    # Strip off any whitespace that might be lurking at the end.
    cols = cols.strip()
    
    # Read everything else into a string.
    dat = f.read()
    
    # There is some whitespace lurking at the end of this string, so get rid of it.
    dat = dat.strip()

col_names = cols.split(",")
baby_lines = dat.split("\n")

babies = []

for baby_line in baby_lines:
    baby_dat = baby_line.split(",")

    # Everything is coming in as a string so we have to make the
    # numbers numbers if we want to do numerical things.

    # Also, some of the names include a `", JR"` substring that we have 
    # to handle. Yikes.

    if len(baby_dat) == 5:
        name = baby_dat[2] + baby_dat[3]
        baby = {"year": int(baby_dat[0]),
            "gender": baby_dat[1],
            "name": name,
            "count": int(baby_dat[-1])}
    else:
        baby = {"year": int(baby_dat[0]),
            "gender": baby_dat[1],
            "name": baby_dat[2],
            "count": int(baby_dat[3])}

    babies.append(baby)

Modules¶

One of the best things about Python is that it is used and developed by a huge number of people. Some of these people write code to do useful things and give the code away for free, posting it on particular sites so that you can use what they've written without writing it yourself.

Most of the code you will use comes in the form of "modules." Python ships with several modules built-into the language. The csv module is a useful one; it provides a set of tools to handle data in comma separated values (csv) format.

To use the module, use the import statement.

In [2]:
import csv

We can use the csv module's DictReader functionality to create a list of dicts without writing as much code, and without having to deal with all of the exceptions we encountered.

In [3]:
with open(filename, "r") as f:
    # Create a `DictReader` object pointing to the csv file
    reader = csv.DictReader(f)
    
    # Create an empty list to hold the data
    baby_data = []
    
    # Iterate over all of the rows in the `DictReader` object we just created
    # and append each to the list of baby data.
    for row in reader:
        baby_data.append(row)
